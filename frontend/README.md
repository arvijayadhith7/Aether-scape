# 🌌 AetherScape

**AetherScape** is a high-performance, intelligent web research workstation designed to transform raw web data into structured insights using AI. 

Built with a stunning **Neubrutalist** aesthetic, AetherScape features a specialized "Turbo Mode" for rapid extraction and a "Zen" interface for distraction-free research.

## 🚀 Key Features

- **Smart Extraction**: Automatically handles layout changes and anti-bot systems.
- **Turbo Mode**: High-speed scraping by skipping human-like delays.
- **Zen Interface**: A clean, modern environment for deep research.
- **AI Insights**: Real-time summarization of extracted data.
- **GSAP Animations**: Fluid, premium UI interactions.

## 🛠️ Technology Stack

- **Frontend**: Next.js, React, GSAP (Animations), Vanilla CSS.
- **Backend**: FastAPI, Pydantic, Playwright.
- **AI**: Gemini/NVIDIA API for extraction and summarization.

## 🏗️ Getting Started

### Prerequisites

- Python 3.10+
- Node.js 18+
- API Keys (NVIDIA/Gemini) in a `.env` file.

### Backend Setup

```bash
# From the root directory
pip install -r requirements.txt
python server.py
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to access the dashboard.
